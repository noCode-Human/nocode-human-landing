# noCode Human Landing

Astro landing site for noCode/Human, including the Tap Tally app page and Convex-backed waitlist.

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Waitlist

The waitlist uses Convex. Set `PUBLIC_CONVEX_URL` in local and hosted environments.
